# 进阶功能

1. 当前界面交互中积木按下即可拖拽，按照如下逻辑进行调整：
    1. 在单个积木块上按下鼠标时，不进行任何选中操作，如果此时执行了拖拽操作（鼠标移动超过mouseMoveThreshold=3个像素），则不选中任何积木块
    2. 定义两块积木相邻：两块积木A,B,存在两个单元格CA,CB；其中CA和CB是相邻的两个单元格。
    3. 定义两块积木相连：两块积木A,B,存在0个或多个积木使得A积木能通过不同积木间的相邻关系找到B。
    4. 定义选中分组：同时选中的多个积木作为选中分组
    5. 定义中心对象：选中分组中的一个积木，显示自身的旋转手柄。拖拽时分组中的积木跟随移动，旋转时整个分组跟随旋转。

    6. 在单个积木块上click(没有鼠标移动或鼠标移动小于mouseMoveThreshold个像素)，如果给积木没有选中或该积木已在选中分组中但是自身不是中心对象时，该积木和该积木相连的所有积木成为选中分组，并将该积木作为中心对象。
    7. 拖拽选中分组中的非中心对象时，分组中的其他对象也会跟随当前积木移动。
    8. 在单个积木上双击（双击事件）仅选中当前积木。

2. 增加检测同一个空间单元格中是否存在多个积木块的逻辑。如果同一个空间中存在多个积木块，左侧列表中对应的缩略图可以通过不同的边框阴影样式进行标识；相应调整选中状态的标识，可以改用不同的背景色进行标识。
